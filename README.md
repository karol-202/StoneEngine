# StoneEngine #
StoneEngine - Open source 3D game engine written in Java

# Features: #
a) *Core*:
   - Objects and component system
   - Keyboard and mouse input
<br>b) *Rendering*:
   - Mesh rendering based on VBO
   - Adjustable camera
   - Materials
   - Textures
   - Shaders system
   - Lights: ambient
<br>c) *Components*:
   - First person controller based on keyboard and mouse input
